|------|------|--------|--------|

## 三、开发者任务

### Backend

#### DEV-B1：项目初始化

| 任务 | 内容 | 产出物 | 依赖 |
|------|------|--------|------|
| DEV-B1.1 | 初始化 Fastify 项目 | 项目骨架、TypeScript 配置 | - |
| DEV-B1.2 | 配置 ESLint + Prettier | 代码规范 | - |
| DEV-B1.3 | 配置 Drizzle ORM + SQLite | 数据库连接、迁移脚本 | - |
| DEV-B1.4 | 初始化数据库 Schema | users, photos, journeys, routes 表 | Drizzle |
| DEV-B1.5 | 编写 seed 脚本 | 预置 G219/G331/G228 路线数据 | Schema |

#### DEV-B2：用户认证

| 任务 | 内容 | API 端点 | 依赖 |
|------|------|---------|------|
| DEV-B2.1 | 发送验证码接口 | POST /api/auth/send-code | 短信服务商 |
| DEV-B2.2 | 验证登录接口 | POST /api/auth/verify | - |
| DEV-B2.3 | 获取当前用户 | GET /api/auth/me | JWT middleware |
| DEV-B2.4 | JWT 签发与验证中间件 | middleware/auth.ts | - |

#### DEV-B3：照片模块

| 任务 | 内容 | API 端点 | 依赖 |
|------|------|---------|------|
| DEV-B3.1 | 上传照片接口 | POST /api/photos/upload | 文件存储目录 |
| DEV-B3.2 | GPS 信息提取 | exif-parser/exifr | - |
| DEV-B3.3 | 获取照片详情 | GET /api/photos/:id | - |
| DEV-B3.4 | 删除照片 | DELETE /api/photos/:id | - |
| DEV-B3.5 | 照片列表（支持筛选） | GET /api/photos | - |

#### DEV-B4：旅程模块

| 任务 | 内容 | API 端点 | 依赖 |
|------|------|---------|------|
| DEV-B4.1 | 创建旅程 | POST /api/journeys | - |
| DEV-B4.2 | 获取旅程详情 | GET /api/journeys/:id | - |
| DEV-B4.3 | 更新旅程状态 | PATCH /api/journeys/:id | - |
| DEV-B4.4 | 获取用户旅程列表 | GET /api/journeys | - |
| DEV-B4.5 | 旅程关联照片 | POST /api/journeys/:id/photos | - |

#### DEV-B5：路线模块

| 任务 | 内容 | API 端点 | 依赖 |
|------|------|---------|------|
| DEV-B5.1 | 获取三条路线信息 | GET /api/routes | seed 数据 |

#### DEV-B6：用户模块

## 六、验收标准

- [ ] 用户可以完成手机号登录
- [ ] 用户可以上传照片并自动提取GPS
- [ ] 用户可以在地图上查看G219/G331/G228三条路线
- [ ] 用户可以创建旅程并关联照片
- [ ] 用户可以在个人主页看到已记录路段
- [ ] 移动端响应式正常显示
- [ ] 页面加载时间 < 3s
- [ ] 无致命安全漏洞（XSS、SQL注入等）
