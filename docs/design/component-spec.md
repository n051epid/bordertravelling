# Bordertravelling MVP UI/UX Design - Component Spec

## 6. 组件规格

### 按钮 (Button)

| 状态 | 背景色 | 文字色 | 边框 |
|------|--------|--------|------|
| Default | #e94560 | #ffffff | none |
| Hover/Press | #c73e54 | #ffffff | none |
| Disabled | #cccccc | #888888 | none |
| Loading | #e94560 | transparent | none + spinner |

尺寸: 高48px, 圆角8px, 内边距0 24px
最小宽度: 120px

### 输入框 (Input)

| 状态 | 边框 | 背景 |
|------|------|------|
| Default | 1px #e0e0e0 | #ffffff |
| Focus | 2px #e94560 | #ffffff |
| Error | 2px #e94560 | #fff5f5 |
| Disabled | 1px #e0e0e0 | #f5f5f5 |

高度: 48px, 圆角6px

### 卡片 (Card)

| 类型 | 背景 | 阴影 | 圆角 |
|------|------|------|------|
| 标准卡片 | #ffffff | 0 2px 8px rgba(0,0,0,0.1) | 12px |
| 路线卡片 | 渐变色 | 0 4px 12px rgba(0,0,0,0.15) | 12px |
| 照片卡片 | #f5f5f5 | none | 8px |

### 照片验证状态徽章

| 状态 | 背景色 | 图标 |
|------|--------|------|
| 待验证 | #ffa500 | 圆点 |
| 验证中 | #2196f3 | 加载动画 |
| 已通过 | #4caf50 | ✓ |
| 未通过 | #f44336 | ✗ |

---

## 7. 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| Mobile | < 640px | 单列，底部导航 |
| Tablet | 640-1024px | 双列，侧边栏 |
| Desktop | > 1024px | 三列，顶部导航 |

---

## 8. 颜色系统

```
Primary:     #e94560 (红色 - 行动号召)
Secondary:   #1a1a2e (深蓝 - 导航背景)
Accent:      #16213e (藏青 - 路线G219)
Success:     #4caf50 (绿色 - 验证通过)
Warning:     #ffa500 (橙色 - 待处理)
Error:       #f44336 (红色 - 错误)
Text:        #1a1a2e (主文字)
Text-light:  #666666 (次要文字)
Border:      #e0e0e0 (边框)
Background:  #f5f5f5 (页面背景)
```

---

## 9. 字体规范

```
主字体: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif

字号:
- 标题H1: 28px / 行高1.2
- 标题H2: 24px / 行高1.3
- 标题H3: 18px / 行高1.4
- 正文:   16px / 行高1.5
- 辅助:   14px / 行高1.4
- 小字:   12px / 行高1.3
```
